# Responsive Mobile Layouts

A mobile-first responsive layout example built with HTML, CSS, and a small JavaScript menu toggle.

## Files

- `index.html` — main page structure and content
- `styles.css` — responsive styling with mobile-first breakpoints
- `script.js` — menu toggle for small screens

## Preview

Open `index.html` in any browser to view the layout. Resize the browser window or use the device toolbar to verify the responsive behavior.

## Notes

- Navigation collapses into a dropdown on narrow viewports.
- Sections stack vertically on mobile and transition to multi-column layouts on larger screens.
- Uses modern CSS features like `clamp()`, CSS Grid, and responsive spacing.
